# Akato 會員制與收費系統規劃

## 1. 會員制核心定位

Akato 的會員制不是單純登入帳號，而是「祝福收藏盒」。

### 品牌定位
Akato 的會員系統核心，不是交易導向，而是把每一次送禮、收禮、收藏祝福語、保存情緒片段，整理成可回來翻看的個人祝福紀錄。它更像一個溫柔的情緒容器，而不是一般帳號中心。

### 情緒價值
* 保存收到的祝福
* 保存送出的心意
* 收藏喜歡的祝福語
* 未來保存花禮影片、數位賀禮、阿哈 Husky 情緒貼圖
* 讓使用者願意回來重新打開自己的情緒紀念

### 與一般電商會員的不同
一般電商會員多半以「消費、折扣、下單」為中心；Akato 會員則以「祝福、回憶、陪伴、收藏」為中心。它不是單純的購物會員，而是一個能持續承載情感內容的祝福收藏盒。會員身份的價值，來自於能否持續保存情緒與關係記憶，而不是只看購買次數。

## 2. 會員層級規劃

### A. 未登入使用者 / 免費訪客

可使用功能：

* 使用送禮頁
* 抽祝福語
* 填送禮人與收禮人
* 產生數位信封連結
* 預覽收禮頁
* 複製祝福連結
* LINE 分享祝福信

限制：

* 不保存紀錄
* 關掉頁面後無法回來管理
* 無法收藏祝福
* 無法保存收到的信

### B. 免費會員

可使用功能：

* 登入帳號
* 保存送出的祝福信
* 保存收到的祝福信
* 收藏喜歡的祝福語
* 查看祝福紀錄
* 儲存常用送禮人名稱
* 未來可收藏阿哈 Husky 貼圖句子

### C. Premium 會員 / 付費會員

可使用功能：

* 免費會員全部功能
* 專屬祝福模板
* 節日祝福包
* 高級信封樣式
* 專屬阿哈 Husky 貼圖
* 花禮紀念頁
* 花禮影片保存
* 生日 / 節日提醒
* 客製化回禮頁
* 未來實體賀禮折扣或限定禮包

## 3. 第一階段不要做什麼

第一階段先不要做：

* 不接金流
* 不做訂閱付款
* 不做複雜權限
* 不做大量後台
* 不做企業會員
* 不做折扣碼
* 不做發票串接
* 不改目前 /letter 信封頁
* 不破壞現有送禮流程

## 5. 建議資料庫模型

### User

欄位：

* id
* email
* displayName
* avatarUrl
* loginProvider
* createdAt
* updatedAt

### Membership

欄位：

* id
* userId
* plan
* status
* startedAt
* expiresAt
* createdAt
* updatedAt

plan 範例：

* free
* premium

status 範例：

* active
* expired
* canceled
* trial

### Gift

欄位：

* id
* senderName
* recipientName
* message
* letterUrl
* createdByUserId
* createdAt
* updatedAt

### GiftCollection

欄位：

* id
* userId
* giftId
* type
* createdAt

type 範例：

* sent
* received
* favorite

### FavoritePhrase

欄位：

* id
* userId
* phrase
* category
* createdAt

### Payment

第一階段只規劃，不實作。

欄位：

* id
* userId
* provider
* providerPaymentId
* amount
* currency
* status
* paidAt
* createdAt

## 6. 金流系統未來流程

使用者選擇升級 Premium
↓
建立付款訂單
↓
導向金流付款頁
↓
付款完成
↓
金流系統打 webhook 到後端
↓
後端驗證付款成功
↓
更新 Membership 狀態
↓
使用者解鎖 Premium 功能

### 特別註明
前端付款成功頁不能直接開通會員。真正開通必須由 webhook 驗證付款後完成。

## 7. 會員功能開發順序

### Phase 1：會員資料與祝福收藏規格
只做規格，不接金流。

### Phase 2：登入與會員資料
加入登入功能，建立 User。

### Phase 3：祝福保存功能
讓登入會員可以保存送出的祝福與收到的祝福。

### Phase 4：Premium 權限狀態
先用假資料或手動設定 plan=premium 測試權限。

### Phase 5：接金流
最後才接付款與 webhook。

## 8. Akato 品牌注意事項

請保留 Akato 的品牌核心：

* 花瓣
* 信封
* 慢慢打開
* 換一句看看
* 阿哈 Husky
* 情緒陪伴
* 日系療癒
* 溫柔留白
* 不是硬推銷
* 不是一般電商感

會員頁也要像「祝福收藏盒」，不要像冷冰冰的後台。
